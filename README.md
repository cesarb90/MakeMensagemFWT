# MakeMensagemFWT
Automação de envio de Mensagem.
A automação busca no Google Drive a planilha correspondente à data atual e aos horários de disparo (9h e 14h). Após localizar o arquivo, o sistema envia as mensagens aos turistas. 
Por fim, realiza um rastreamento identificando os destinatários que receberam a mensagem e os que não receberam, registrando uma atualização com o motivo da falha no envio.
